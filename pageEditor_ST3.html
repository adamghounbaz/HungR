I Was Involved In Making The Following Web Pages:

User Profile:

The user profile web page allows users to create an account with all their information such as qualifications, address and more. Users will be able to see all the information they have inputted during the account creation and at the very end, they will also be able to edit their inputs. There is a progress bar that dynamically updates as the user fills out their information.

I have used a lot of java script throughout the website to save user's answers and dynamically output them to the user as they create their account. I have also used css to style the website and the progress bar.

Team Page:

The team page includes pictures of all members of the team that designed this website. It includes their names and when the cursor is hovered over the images, the team member's role in the website's production and what webpages they've designed is shown.

I've used css throughout the website in order to style it and create feedback to the user when their cursor is hovered over an image. I also used css to display more information of the team member that the cursor is hovered over to the user.

Content Page (Ending World Hunger):

The content page includes various images and paragraphs explaining how our organization will help to end world hunger. There are various links in the webpage that allows users to quickly scroll to different subheadings

Css has been used throughout the webpage to both style and resize the images and to create a navbar to allow users to easily skip to different subheadings by making the nav bar permanently fixed on the top of the screen.
